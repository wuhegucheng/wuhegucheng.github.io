# 虚拟机网络模式

  • 桥接模式

    • 物理机网络与虚拟机网络共享
    • 物理机与虚拟机需要在同一个网段
    • 可用地址 0-255（除 0、1、2：网关、255）
    • 耗费 IP（IP 不能冲突）

  • NAT 模式

    • 路由器将一个公有地址转换为多个私有地址（NAT 动态网络地址转换）
    • DHCP ：动态获取 IP 地址

  • 仅主机模式

    • 虚拟机之间相互通信，无法上外网（局域网）
