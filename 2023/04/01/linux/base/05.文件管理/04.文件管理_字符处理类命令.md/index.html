# 字符处理类命令

# 字符处理命令 - 排序 sort

选项 说明
-t 指定分隔符
-k 指定按照某一列，进行排序
-n 按照阿拉伯数字排序
-r reverse 倒叙排序

# 去重命令 uniq

如果文件中有多行完全相同的内容，希望能删除重复的行，同时还可以统计出完全相同的行出现的总次数，那么就可以使用 uniq 命令解决这个问题，必须配合 sort 使用

选项 说明
-c 计算重复的行

# 文件内容截取命令 cut

选项 说明
-d 指定分隔符
-f 取列，选择要打印的列内容
-c 按照字符来取内容

# 文件统计命令 wc

选项 说明
-l line 行，统计行数
-w word 单词，统计单词数
-c char 字符，统计字符数
