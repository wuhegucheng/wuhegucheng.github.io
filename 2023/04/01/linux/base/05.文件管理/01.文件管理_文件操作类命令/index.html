# 文件复制 cp

选项 说明
-p 保持文件属性
-r 递归复制，复制目录及目录下的所有文件
-a 相当于 - rp，既可以拷贝目录，又可以保持文件属性
-i 提示是否覆盖（cp 命令默认使用 "cp -i"）
-v 显示拷贝的过程

# 删除命令 rm

选项 说明
-i 提示是否删除（rm 命令默认使用 "rm -i"）
-r 递归删除
-f 强制删除 force
-d 只能删除空目录
