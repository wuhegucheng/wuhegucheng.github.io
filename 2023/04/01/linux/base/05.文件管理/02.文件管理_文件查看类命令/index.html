# 文件查看类命令

# cat 命令

选项 说明
-n number 打印行号，显示行号
-E end ，查看文件每一行的结尾，以 $ 符表示
-T Tab，查看文件中每一行内容是否含有制表符。以 ^I 表示一个制表符
-A 既显示制表符，又显示文件的结尾

# more、less 命令

# head 命令

选项 说明
-n 指定查看文件开头的前 n 行，n 可以省略

# tail 命令

选项 说明
-n 指定查看文件末尾的后 n 行，n 可以省略
-f follow，跟踪，动态跟踪一个文件的末尾

# 扩展

catmorelessheadtail
